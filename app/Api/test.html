The Hard Parts of UI Development

UI tools like React, Angular, and Vue help, but they can be hard to use if you don't know what problems they solve. Learn how JavaScript interacts with browser features like the DOM and HTML parser. See how UI code remains performant at scale while maintaining the reactivity between the underlying data and the DOM elements. This performance is achieved by reconciling virtual DOM updates with the use of state hooks and a diffing algorithm.
Other under-the-hood UI rendering topics you’ll learn are:

- Understanding the role WebIDL and WebCore play in browser rendering
- Updating DOM elements with one-way data binding
- Creating a predictable flow of data to the view
- Building composable UIs in JavaScript with a Virtual DOM
- Optimizing rendering performance with DOM Diffing
