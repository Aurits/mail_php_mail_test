    Industry news and updates

    Analysis by Jens Oliver Meiert reveals that none of the 100 most visited sites globally are fully conformant with the HTML spec, so maybe you don't need to feel too bad about your own site.

    Vercel unveiled Next.js 14 at Next.js Conf. The headline feature is that support for Server Actions is now considered stable, so you can partake in some fun mixing of frontend and backend development, thanks to 'use server'.
    FYI we already have two courses that cover server actions Introduction to Next.js 13+, v3 and Build an AI-Powered Fullstack Next.js App, v3

    From the folks behind the popular State of JavaScript and State of CSS surveys comes yet another survey: The State of React 2023. As with their other surveys, you get to test your knowledge along with answering questions, so it can be quite fun to do.
    And if you're looking to dive into all those parts of React you don't know, our React learning path is a great place to dive in!

    Node is experimenting with automatically detecting and running ESM syntax with Node v21.1, the latest release of the current branch. In related news, Node v20 just became the latest active LTS release line as of Node v20.9.0.

    You can now add horizontal rules to a select element, a handy addition to help visually seperate items in a drop down list. It works in Chrome 119 and Safari 17 right now. A beautiful coming together of two of the oldest HTML elements.

    ✨ Here's some key releases you may want on your radar: Netlify announced their new 'composable web platform', Astro 3.4 is out now, Docusaurus hit 3.0, TanStack Query v5 was released, and npm alternative Yarn is now at 4.0.

    Useful projects

    CSS Box Shadows Generator is a tool for working with CSS shadows, including experimenting what happens when you layer shadows on top of each other. This is a great way to make some striking looking elements with very little CSS.

    background-removal-js is a neat npm package that can add handy image background editing functionality to your web app without the need for any third party services.

    FormsLab is an open-source survey creator, a la Typeform.

    React Magic Motion is a library that sits on top of Framer Motion and makes it easier to add basic animations to your React components.

    ApexCharts is a JavaScript charting library that not only produces good looking charts and similar data visualizations, but also has lots of demos and samples to be inspired by.

    This tool from DebugBear analyzes CSS to spot bloat and large CSS rules.
